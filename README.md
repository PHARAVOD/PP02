# PP02 — Производственная практика (Разработка программного обеспечения)

## Описание

Данный репозиторий содержит материалы производственной практики ПП.02 по направлению 09.02.07 «Информационные системы и программирование»

**Тема:** Разработка автоматизированной системы управления пунктом выдачи заказов (АСУ ПВЗ)

**Организация:** ИП Платонов Олег Владимирович

**Стек технологий:**
- Python / Django
- PostgreSQL / SQLite
- JavaScript (Vanilla)
- HTML5 / CSS3
- PlantUML
- Git / GitHub

---

## Содержание

- `docs/` — документация, отчёт, ТЗ, UML-диаграммы
- `database/` — скрипты создания и заполнения БД
- `backend/` — серверная часть (Django)
- `frontend/` — клиентская часть (HTML, JS, CSS)
- `scripts/` — скрипты автоматизации (Python)
- `tests/` — модульные и интеграционные тесты

---

## Функциональность

Авторизация и разграничение прав доступа
Импорт заказов из Excel
Поиск заказов (по номеру, телефону, трек-номеру)
Адресное хранение и назначение ячеек
Выдача заказов с печатью чека
Оформление возвратов
Инвентаризация
Формирование отчетов
Резервное копирование
 Мониторинг системы
